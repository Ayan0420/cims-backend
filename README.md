# CIMS

Fullstack application for Comtech Information Management System written Typescript with NestJS and ReactJS

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

#build Nest app and client React app
$ npm run build:all
```
